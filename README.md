# 동물 슈팅 게임 🚗💥🐾

HTML5 Canvas + JavaScript로 제작된 12지신 동물 슈팅 게임입니다.  
Python(pywebview)으로 네이티브 창에서 실행되며 PyInstaller로 EXE로 배포할 수 있습니다.

---

## 게임 소개

| 항목 | 내용 |
|------|------|
| 장르 | 수직 스크롤 슈팅 (Vertical Shooter) |
| 언어 | HTML5 · CSS3 · JavaScript · Python |
| 단계 | 1~12단계 (12지신 동물) |
| 목표 | 각 단계의 동물을 처치하여 12단계 클리어 |

| P | 일시정지 |

---

## 실행 방법

### 방법 1 — 브라우저에서 바로 실행

```
index.html 파일을 브라우저로 열기
```

### 방법 2 — Python으로 실행 (네이티브 창)

```bash
pip install -r requirements.txt
python launcher.py
```

### 방법 3 — EXE 빌드 후 실행

```bash
build.bat
# dist\동물슈팅게임.exe 실행
```
